Hello

Huzjak-programming is a full stack technology provider from the North of Croatia, specializing in the development of commercial mobile applications and web solutions. We can utilize our years of experience in web development to help you make the best components or whole solutions for your web and mobile projects. We can also work with your team to have those completely customized to your needs.

What can we do for your company?

Mobile Applications

Development of iOS (iPhone, iPad), Android and Windows mobile applications (smartphones, tablets). We create mobile applications with standard web technologies (HTML, CSS, JavaScript) and use PhoneGap to convert them to native mobile apps.

Mobile Components

Making mobile-optimized components for billing, secure transactions, and certificates, as well as user management between platforms.

Websites

Based on your design or ready-made templates we quickly and efficiently transform it into a beautiful, fully-functioning and responsive website which will work great on all devices.

Web Applications

Development of various dynamic web applications and solutions tailored to your needs and wishes. This includes the creation of a database, server-side software as well as visually pleasing and interactive frontend.

Web Components

Instead of committing to a big web application, you can choose to upgrade your existing web app with just some new modules or start a new app with only the necessary components.

Custom Development

We're able to furnish you with a wide range of tailor-made technical solutions, so if you have a specific requirement - please get in touch!

About Us
